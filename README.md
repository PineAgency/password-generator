# Advanced Password Generator

A modern, highly customizable, and secure password generator built for the web.

## Features

- **Robust Generation:** Create secure passwords up to 128 characters long with fine-grained control over character sets (uppercase, lowercase, numbers, symbols).
- **Passphrase Mode:** Generate memorable diceware-style passphrases using a built-in dictionary and customizable separators.
- **Advanced Options:** Toggle options to exclude ambiguous characters (like `l`, `1`, `O`, `0`) and prevent duplicate characters to improve readability and usability.
- **Strength Estimation:** Real-time entropy calculation and estimated crack-time feedback help you ensure your passwords are truly secure.
- **Recent Passwords Sidebar:** A dedicated, responsive sidebar automatically tracks the last 5 generated passwords so you never lose track of a good one.
- **Dark Mode:** A sleek, automatically saved dark mode theme for comfortable viewing.
- **Privacy First:** 100% client-side logic. Passwords are generated locally in your browser, and nothing is ever sent to a server.

## Built With

- **HTML5 & CSS3**
- **JavaScript (ES6+)**
- **Tailwind CSS** (via CDN) for rapid styling
- Fonts: **Inter** for UI text and **JetBrains Mono** for password displays
